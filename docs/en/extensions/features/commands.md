# Commands System

The extension registers the following core commands for quick access to various features.

- **View Category**
  - `monoco.openKanban`: Open Task Kanban Webview.
  - `monoco.openSettings`: Open extension configuration page.
  - `monoco.openWebUI`: Open Monoco Web interface in browser.

- **Action Category**
  - `monoco.createIssue`: Open interactive interface to create new Issue.
  - `monoco.runAction`: Start SOP action selection and execution flow.
  - `monoco.refreshEntry`: Force refresh Kanban data and action list.

- **Auxiliary Category**
  - `monoco.showAgentActions`: Show recommended actions available in current context.
  - `monoco.viewActionTemplate`: Preview Prompt template content of selected action.
