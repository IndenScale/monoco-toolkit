# Configuration & Dependencies

## 5.1 Extension Configuration

- **`monoco.webUrl`**
  - **Type**: String
  - **Default**: `http://127.0.0.1:8642`
  - **Purpose**: Defines access address for external Web UI.

## 5.2 External Dependencies

- **Monoco CLI**
  - **Usage**: Core business logic execution (Lint, Update, Index).
  - **Requirement**: Must be accessible in system PATH.

- **Node.js Runtime**
  - **Usage**: Running Language Server.
