# Monoco Toolkit

Agent Native CLI for Monoco Development.

## Install

```bash
pip install -e .
```

## Dual Frontend

- **Human Mode**: Rich text output (Default)
- **Agent Mode**: Set `AGENT_FLAG=true` for compact JSON output.
