# DevEx

## 定义
致力于提升 Monoco Toolkit 使用者（包括人类开发者与 Agent）工作效率、交互体验与满意度的领域。

## 职责
- **Tooling Experience**: 优化 CLI 命令的交互反馈、错误提示与自动补全。
- **IDE Integration**: VSCode 插件、LSP (Language Server Protocol) 支持与编辑器集成。
- **Linting & Formatting**: 代码风格检查、Issue 规范校验与自动修复工具。
- **Workflow Optimization**: 简化常用操作流程，减少重复性劳动。
