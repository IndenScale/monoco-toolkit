# Monoco Toolkit

适用于 Monoco 开发的 Agent 原生命令行工具 (CLI)。

## 安装

```bash
pip install -e .
```

## 双模式前端 (Dual Frontend)

- **人类模式 (Human Mode)**: 富文本输出（默认）。
- **Agent 模式 (Agent Mode)**: 设置 `AGENT_FLAG=true` 以获得紧凑的 JSON 输出。
